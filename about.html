About Velvet Vault

VIP Noir Slots Club

Velvet Vault is a slots-only experience built around a hybrid architecture: Firebase Cloud Functions hold wallet authority, Render computes outcomes, and the frontend stays fast and static.

What It Is

Members move from the VIP hub into the Slots Lobby, choose a machine, and play through an atomic reserve-spin-settle flow.

How It Works

Balances are authoritative in Firebase. The slot server never edits money. The frontend only orchestrates the secure flow.

Why Slots Only

The current product focus is clarity, control, and polish around a single game category without table-game drift.
